### NestInsights — Consumer Intelligence Platform &nbsp;·&nbsp; [View Repo](https://github.com/zain-ul-abideen-1/Nestinsights-Consumer-Intelligence-Platform) &nbsp;·&nbsp; Live

A production-grade, full-stack consumer intelligence platform built specifically for e-commerce businesses. Integrates 5 open-source Hugging Face models alongside custom-trained ML models (TF-IDF + LinearSVC, BERT, BiLSTM) achieving 89% classification accuracy on 10,500+ samples scraped from Trustpilot and major review platforms.

**Core capabilities:**

- **Deep AI Analysis System** — multi-layer sentiment analysis, fake review detection, aspect-level feedback extraction, and AI-generated executive reports surfacing actionable product and brand insights
- **Global Market Intelligence** — on-demand analysis of product demand shifts driven by global events, geopolitical changes, and market conditions, with real-time impact scoring across product categories
- **Business Intelligence Hub** — full visual analytics suite including trend graphs, sentiment distribution charts, rating breakdowns, competitor comparison charts, revenue impact formulas, and regression-based forecasting
- **Support Ticket Classifier** — automated ticket routing and prioritization using NLP classification
- **Real-Time AI Assistant** — Groq-powered conversational agent with live read/write access to platform data
- **Tech:** MERN Stack · BERT · BiLSTM · TF-IDF + LinearSVC · 5x Hugging Face Open-Source Models · Groq API · Python · REST APIs · Railway
